Mac Outlook Sounds

I got distracted...

Aug 25, 2016 by td

I have more than one Mac, and get email on both. That sound gets annoying. First I turned it off on both, then found I missed the notification at times.

When I was working on the backup laptop I liked getting notified, the main Mac is on all the time and I left that notification off. But the personal backup I decide to put a new sound so I'd know MY email.

When I dont want to hear it, I shut the lid.

As it turns out, you can change and add your own sounds and really you can't. So I did.

Download this file then unzip it and move it to your library sound file area as shown. The file is copied from Microsoft's Entourage product.

You can then muck about with the contents as you wish. Just edit he PLIST file, it contains XML for the sound set, and the files are expected in the parent folder.

The file name extension apparently needs to be as shown in the image.

For me the folder was /Users/tdavis/Library/Group Containers/UBF8T346G9.Office/Outlook/Outlook Sound Sets

I suspect the gibberish at the beginning is some sort of release code that may be unique to you.

MVC Architectures

My book due out this summer.

March 22, 2016 by td

Model-View-Controller (MVC) is nothing new really. Separation of data access, presentation and data model meant that applications could mutate and adjust to changing data and presentment.

Like many software concepts its never as easy as we'd like, but still this yeilds manageable and achieveable architectures.

My book examines some popular architectures and compares the benefits and gotchas of each.

  • EmberJS
  • AngularJS 1.4
  • AngularJS 2.0
  • React
  • Backbone

Each of these has a sweet spot. I make conclusions and recommendations for each and suggest where the WWW needs to go in the next MVC architectures.

I plan on publishing chapters as they are editted on my blog.
